# Advice Generator

Advice Generator is a web application that fetches and displays random advice quotes from an external API. Built with HTML, CSS, and JavaScript, it offers a clean, responsive interface for users to receive quick inspiration or guidance with a single click.

## Demo
[**Try Advice Generator Live**](https://ismail-advice.netlify.app/)

## Features
- **Random Advice**: Fetches unique advice quotes from an API with each click.
- **Responsive Design**: Adapts seamlessly to desktop and mobile devices.
- **Modern UI**: Sleek, user-friendly design with smooth animations.
- **Lightweight**: Minimal code for fast loading and performance.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/ismail9700/advice-generator-app.git
   ```
2. Navigate to the project directory:
   ```bash
   cd advice-generator-app
   ```
3. Open `index.html` in a browser or serve it using a local server:
   ```bash
   # Using live-server (Node.js)
   npm install -g live-server
   live-server
   
   # Or using Python
   python -m http.server 8000
   ```

## Usage
- Open the app in a browser.
- Click the button to generate and display a new piece of advice.
- Customize the HTML, CSS, or JavaScript to modify styling or functionality as needed.

## Technologies Used
- **HTML5**: Structure for the web app.
- **CSS3**: Styling with modern design techniques.
- **JavaScript**: Handles API calls and dynamic content updates.
- **External API**: Fetches random advice quotes (e.g., Advice Slip API).
